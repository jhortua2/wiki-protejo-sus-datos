## 📋 Sobre este proyecto

Wiki interactiva que analiza la arquitectura de red más segura para una empresa de almacenamiento en la nube, desarrollada como Actividad Evaluativa del Eje 02 de **Sistemas Operativos II**.

---

## 📚 Temas cubiertos

- 🔗 Red Peer-to-Peer — propuesta inicial
- 🔐 Riesgos de seguridad de la información
- 🖥️ Red Cliente-Servidor — condiciones, ventajas y desventajas  
- 🏗️ Arquitectura de red propuesta completa
- 👥 Grupos de usuarios y perfiles de acceso
- 📌 Conclusiones y recomendaciones
